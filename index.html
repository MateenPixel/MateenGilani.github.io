Flag Feature Development Tool

While interning at Radian as a DevOps Software Developer, I developed a Flag Feature tool that enables developers to toggle features on/off, helping in controlled rollouts, A/B testing, and making the application more robust and adaptable to changes.

×

During this project, I conducted all work on a local machine, including the development of Figma wireframes. This experience was invaluable, providing me with deep insights into the workflows and interactions of different teams within a development office. As a comprehensive full-stack project, it greatly enhanced my skills in a variety of tools and technologies, including Visual Studio Code (VSC), Postman, Figma, GitHub, Docker, DBeaver, PostgreSQL, HTML, CSS, and ReactJS. Unfortunately, because the project was completed locally, I couldn't add it to my GitHub or portfolio, leaving me with only the knowledge and experience gained from the process.
